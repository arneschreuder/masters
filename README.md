# Environments

## Activate Environment

```sh
source .venv/bin/activate
source .env
```

## De-activate Environment

```sh
deactivate
```

# How to use?

## Prerequisites

```sh
pip install -r requirements.txt
```

## Run

```sh
python main.py
```
